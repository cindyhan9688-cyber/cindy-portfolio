HTML → 长图 / PDF 工具

⏱ 2026-07-14Python · Playwright● 已上线，日常在用

任何网页报告，一条命令变成「能直接发群的高清长图」+「不被切碎的单页长版 PDF」。

GitHub 代码仓库 → 返回

产品思路

为什么做、怎么取舍、做完带来什么。

WHY

解决真痛点

网页报告分享一直很别扭：发链接对方打不开，截屏太糊，浏览器打印 PDF 又被切成十几页碎片。我要的是"一条命令，直接拿到能发微信群的成品"。

HOW

克制的设计

单文件零配置；优先复用本机 Chrome/Edge 内核，绕开国内下载浏览器内核卡死的坑；PDF 按内容实际高度输出"单页长版"而非 A4 分页；只留 --dark / --width / --scale 三个参数，覆盖 95% 场景。

RESULT

成为基础设施

现在是我个人工作流的底层工具：旅游攻略、分析报告、数据周报全部一键出图；2 倍清晰度长图直接发群不糊，PDF 永远一页到底。

使用方式

一条命令，两个成品。

python html_export.py report.html
OK PNG: report.png  (158 KB)
OK PDF: report.pdf  (93 KB)
  • 内核回退链 ：chrome → msedge → chromium 依次尝试，无需 playwright install 也能跑
  • emulate_media("screen") ：PDF 保留屏幕配色和背景，而不是惨白的打印样式
  • 按内容高度出单页 PDF ：读取页面实际 scrollHeight 作为纸张高度，长报告不分页
  • device_scale_factor=2 ：长图默认 2 倍清晰度，微信里放大看文字仍然锐利
